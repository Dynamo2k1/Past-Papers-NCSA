# Past Papers — National Cyber Security Academy (NCSA)

This repository is a simple collection of past exam papers, practice questions, and related study materials for the National Cyber Security Academy (NCSA). The materials are provided to help students study and prepare for assessments.

## Contents
- Past exam papers and practice questions organized by subject and year.
- Any supporting materials (mark schemes, instructions) where available.

## How to use
- Browse the folders in this repository to find papers by subject and year.
- Download individual files or clone the repository:
  ```
  git clone https://github.com/Dynamo2k1/Past-Papers-NCSA.git
  ```

## How to contribute
If you have additional past papers or improvements to add:
1. Fork this repository on GitHub.
2. Clone your fork and create a branch:
   ```
   git clone https://github.com/<your-username>/Past-Papers-NCSA.git
   cd Past-Papers-NCSA
   git checkout -b add-new-papers
   ```
3. Add files, commit, and push your branch:
   ```
   git add .
   git commit -m "Add past papers for <subject> <year>"
   git push origin add-new-papers
   ```
4. Open a pull request from your branch to this repository's `main` branch with a short description of your changes.


That's it — feel free to fork, add papers, and send a pull request.
